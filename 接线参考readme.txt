
 Music Spectrum - ESP32-S3 N16R8 引脚分配参考


芯片: ESP32-S3 N16R8 (16MB Flash / 8MB PSRAM)
屏幕: ST7796 320x320 (4线 SPI)
麦克风: INMP411 PDM 数字麦克风 (I2S)

------------------------------------------------------------
 ST7796 LCD (4线 SPI, SPI2_HOST)
------------------------------------------------------------
 GPIO12  →  SCLK   (SPI 时钟)
 GPIO11  →  MOSI   (SPI 数据输出)
 GPIO10  →  DC     (数据/命令选择)
 GPIO9   →  CS     (片选, 手动控制)
 GPIO13  →  RST    (面板复位)
 GPIO14  →  BL     (背光, 高有效)

------------------------------------------------------------
 INMP411 麦克风 (I2S_NUM_0, 标准 Philips 模式)
------------------------------------------------------------
 GPIO41  →  BCLK   (I2S 位时钟)
 GPIO42  →  WS     (I2S 字选/LRCK)
 GPIO39  →  DATA   (I2S 数据输入)

------------------------------------------------------------
 音频 + FFT 参数
------------------------------------------------------------
 采样率:     16000 Hz
 FFT 点数:   1024 点 → 512 频段
 显示柱子:   32 根 (对数频段合并)
 平滑上升:   α = 0.4
 平滑下降:   α = 0.15
 峰值保持:   800ms

------------------------------------------------------------
 LVGL 显示参数
------------------------------------------------------------
 分辨率:     320 x 320
 色深:       16-bit (RGB565, 字节交换)
 双缓冲:     320 x 40 每块 (PSRAM)
 刷新周期:   16ms (~60Hz)

------------------------------------------------------------
 FreeRTOS 任务分配
------------------------------------------------------------
 Core 0: audio_fft_task   (8192B, prio 5)  I2S采集 + FFT计算
 Core 0: ui_update_task   (4096B, prio 3)  频谱平滑 + LVGL刷新
 Core 1: display_task     (8192B, prio 2)  LVGL渲染循环

------------------------------------------------------------
 接线快速检查清单
------------------------------------------------------------
 [ ] GPIO12 ↔ LCD SCLK
 [ ] GPIO11 ↔ LCD MOSI (SDA)
 [ ] GPIO10 ↔ LCD DC
 [ ] GPIO9  ↔ LCD CS
 [ ] GPIO13 ↔ LCD RST
 [ ] GPIO14 ↔ LCD BL
 [ ] GPIO41 ↔ I2S BCLK
 [ ] GPIO42 ↔ I2S WS (LRCK)
 [ ] GPIO39 ↔ I2S DATA (INMP411 DOUT)
 [ ] VCC ↔ 3.3V
 [ ] GND ↔ GND
